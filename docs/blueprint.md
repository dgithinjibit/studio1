# **App Name**: CurriculumAI

## Core Features:

- Query Processing: Accepts trainee questions through a chat interface.
- Contextual Retrieval: Uses a vector database or simulated retrieval system to select relevant DPTE course document chunks.
- AI-Powered Response: Generates comprehensive answers grounded in the retrieved documents using Gemini API. Instruct Gemini, as a tool, to only incorporate material from the provided context, rather than any of its other knowledge.
- Curriculum Integration: Offers curriculum-aligned and contextually relevant responses.
- Futuristic Chat UI: Presents a user-friendly, visually engaging interface suitable for 20-year-old teens.
- Submit Assessment: Allows the teachers to be assessed according to responses made to a query.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) - Hex #3399FF - A vibrant, tech-forward blue evoking trust and intelligence.
- Background color: HSL(210, 20%, 95%) - Hex #F0F8FF - A light, desaturated blue for a clean, modern backdrop.
- Accent color: HSL(180, 60%, 40%) - Hex #33B2B2 - An analogous teal with high contrast, for interactive elements and emphasis.
- Headline font: 'Space Grotesk', a modern sans-serif. Body font: 'Inter', a grotesque-style sans-serif. These will be used for a clean, professional look.
- Utilize a set of minimalist line icons related to education, teaching, and AI to provide a consistent visual language.
- Clean, card-based layout for document chunks with clear separation and hierarchy.
- Use subtle loading animations and smooth transitions to enhance the user experience.
